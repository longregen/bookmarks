Welcome to Bookmark RAG

Your AI-powered bookmark manager

What is Bookmark RAG?

Bookmark RAG captures web pages and uses AI to generate searchable question-answer pairs, enabling semantic search across your bookmarks. Never lose track of important information again.

Key Features

  • Semantic Search: Find bookmarks by meaning, not just keywords
  • AI-Powered Q&A: Automatically generates questions and answers from your saved pages
  • Local Storage: All bookmarks stored securely in your browser
  • Flexible API Support: Works with OpenAI and compatible providers

Get Started

To use Bookmark RAG, you need to configure an API endpoint for AI processing. This is required for generating embeddings and semantic search capabilities.

Privacy & Security

All bookmarks are stored locally in your browser using IndexedDB. Only the extracted text content is sent to your configured API endpoint for processing. Your API key is stored securely in your browser and never shared with third parties.
