Features

One-Click Capture
Save URLs, titles, and full page content with a single click or keyboard shortcut (Ctrl+Shift+B / Cmd+Shift+B)
Intelligent Content Extraction
Automatically converts HTML to clean Markdown using Mozilla's Readability engine
AI-Generated Q&A Pairs
LLM generates 5-10 smart question-answer pairs about each bookmark for better searchability
Semantic Search
Search your bookmarks by meaning and context, not just keywords. Find what you're looking for even if you don't remember the exact words.
Bulk Import
Import hundreds of URLs at once. Process 100 bookmarks in 2-3 minutes with parallel fetching.
Export & Backup
Export your entire collection as JSON anytime. Your data, your control.

How It Works

1. Capture — Click the extension or press the shortcut. Full page HTML is saved locally.
2. Extract — Main content is extracted and converted to clean, readable Markdown.
3. Generate — Your configured LLM creates Q&A pairs about the content.
4. Embed — Each Q&A pair is converted to vector embeddings for semantic search.
5. Search — Query with natural language. Results ranked by semantic similarity.

Privacy First

> All bookmarks stored in your browser's IndexedDB — never uploaded to any server.

> Only extracted Markdown is sent to your configured API for processing.

> You control the API endpoint — use OpenAI, local models (Ollama, LM Studio), or any compatible service.

> Export your data anytime. No lock-in, no surprises.

Getting Started

1. Download and install the extension for your browser
2. Click the extension icon and go to Settings
3. Configure your API endpoint and models (OpenAI or compatible)
4. Click "Test Connection" to verify setup
5. Start saving pages and searching semantically!
